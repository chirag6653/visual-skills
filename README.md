## 🚀 Using Your Skills

The folder you extracted contains several subdirectories. Each directory corresponds to a specific AI tool or model. To use these prompts:

1. Open your AI agent or code editor (Cursor or Windsurf are recommended).
2. Open the prompt file inside the visual-skills folder.
3. Copy the prompt text to your clipboard.
4. Paste the text into your AI tool’s chat window.
5. Replace the bracketed text with your specific project details.
6. Send the message to generate your image or video.
